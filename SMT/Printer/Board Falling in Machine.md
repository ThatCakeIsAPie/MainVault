Problem:
After the printer, the board is falling into the shuttle conveyor.

Root Cause(s):
1. Sensor was not detecting board
2. Conveyors are not fully aligned/too wide

Solution(s):
1. Adjust/clean the sensor. Verify the sensor is working correctly via the sensor program
2. Verify the conveyor is aligned properly by moving the board between the machines and checking if both sides of the conveyor touch the board and are flat. For additional information, reference [[Board Falling in Machine]]
3. Camera needs cleaning
